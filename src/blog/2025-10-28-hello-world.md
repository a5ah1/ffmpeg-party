Welcome to the relaunched ffmpeg.party! This site has been completely rebuilt from the ground up as a cleaner, more organized resource for FFmpeg tools and knowledge.

## What is ffmpeg.party?

This is a personal project where I share tools and guides for FFmpeg that I've created for my own use. Rather than keeping them to myself, I figured they might be useful to others dealing with video encoding challenges.

## What's New

The site has been restructured with improved organization and several new additions:

**Tools**
- Visual **Cropper** for generating FFmpeg crop filters
- **Anamorphic Crop Calculator** for handling anamorphic video properly
- Updated encoding wizards (VP9, AV1, x264, x265) with better UI and isolated settings
- Duration calculator and other utilities

**Guides**
- Comprehensive **x264** encoding guide with quality recommendations
- **x265** guide for HEVC encoding
- New **AV1** guide covering SVT-AV1 parameters and settings

**Infrastructure**
- Better organized navigation
- Cleaner, more consistent interface
- Improved tool persistence (settings save per-tool instead of globally)

## What's Next

The roadmap for the near future includes:

- **Batch encoding script generator** - Tools to help create scripts for encoding multiple files with consistent settings
- **Filter helpers** - Utilities for working with complex filters, like frame rate doubling with smooth interpolation using minterpolate
- Additional guides and documentation as needed

This blog will occasionally feature updates about new tools, encoding tips, and other FFmpeg-related notes. No set schedule—just updates when there's something worth sharing.
